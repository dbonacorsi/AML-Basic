# AML-Basic

Material for the AML-Basic hands-on by DB and LC.

## Contents

- **Tutorials with DB**: contained in root repo path

- **Tutorials with LC**: Python libraries introduction and fundamentals
    - Located in `python-libraries-intro` folder
    - Covers essential data science and ML libraries

## Getting Started

### Recommended Setup
We suggest running notebooks through **Google Colab** to avoid local environment configuration:
- No installation required
- Pre-configured Python environment
- GPU access available
- Refer to instructions provided during lectures

### Local Setup (Alternative)
If you prefer to run locally, ensure you have Python 3.x installed with required dependencies listed in `requirements.txt`.
